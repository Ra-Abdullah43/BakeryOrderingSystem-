# Bakery Ordering System

It is a simple system for the bakery owners/managers to manage their bakery to take orders from the customers. It is either a customized cake or just a ready-made cake from the bakery and if it is a customized cake, the customer can ask for delivery. The manger can see the delivered orders , the local orders and all orders.

## Project workflow

Use any appropriate IDE for java to run this project.
